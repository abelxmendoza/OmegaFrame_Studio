# Placeholder Thumbnail Image

This directory should contain a placeholder thumbnail image at:
`/public/images/placeholder-thumbnail.png`

The image should be:
- 1280x720 resolution (16:9 aspect ratio)
- Dark theme with purple accents matching OmegaFrame Studio branding
- Simple design that indicates "no thumbnail available"

You can create this image or use a default placeholder. The `ProjectThumbnail` component will automatically use this as a fallback when:
1. No thumbnail URL is provided
2. The thumbnail image fails to load

Example design ideas:
- Dark background (#0b0b0f) with purple glow (#7938ff)
- Simple icon or logo
- Text: "No Thumbnail" or similar
